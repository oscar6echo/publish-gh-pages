Static Content

Hosted on GitHub Pages

The urls are built as follows:

Enterprise GitHub
https://[hostname]/pages/[username]/[repo-name]/[dir-name]/[filename]
GitHub.com
https://[username].github.io/[repo-name]/[dir-name]/[filename]
For example:
  • https://github.com/pages/oborderi/publish-static-content/sample.html
  • https://github.com/pages/oborderi/publish-static-content/new_content_1.html
  • https://github.com/pages/oborderi/publish-static-content/new_content_2.html
